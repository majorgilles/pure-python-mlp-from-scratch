# Bike rentals teaching datasets

`bike_rentals.csv` is a teaching dataset prepared from the UCI Bike Sharing Dataset hourly data.

The goal is not to create a statistically perfect benchmark. The goal is to give this course a real regression dataset that is still simple enough for a pure-Python neural network lesson.

## Source

Original dataset:

- UCI Bike Sharing Dataset: <https://archive.ics.uci.edu/dataset/275/bike+sharing+dataset>
- Column description mirror: <https://github.com/pgebert/bike-sharing-dataset/blob/master/README.md>

The preparation script downloads the UCI zip file and reads `hour.csv`.

## Row-selection rule

The dataset is generated by `scripts/prepare_bike_rentals_mini.py`.

The default rule is now:

1. Read the UCI hourly dataset, `hour.csv`.
2. Keep every available hourly row.
3. Write only the teaching columns listed below.

This creates `data/bike_rentals.csv` with the full available UCI hourly data in the simplified teaching schema.

The old balanced mini dataset can still be recreated by sampling rows per hour:

```bash
python scripts/prepare_bike_rentals_mini.py --rows-per-hour 10 --output data/bike_rentals_mini.csv
```

That mini rule uses Python's standard-library `random` module with seed `8`, selects `10` rows from each hour group, and writes `24 * 10 = 240` rows.

## Columns

| Column | Meaning |
| --- | --- |
| `hour` | Original hour of day from UCI `hr`, from `0` to `23`. |
| `hour_scaled` | `hour / 23`, so the hour is written between `0.0` and `1.0`. |
| `temperature` | UCI `temp`. UCI already stores this as a normalized-style weather value. |
| `humidity` | UCI `hum`. UCI already stores this as a normalized-style weather value. |
| `wind_speed` | UCI `windspeed`. UCI already stores this as a normalized-style weather value. |
| `working_day` | UCI `workingday`, where `1` means the day is a working day and `0` means it is not. |
| `rental_count` | UCI `cnt`, the original bike rental count for that hour. |
| `target_scaled` | `rental_count / 1000`, the model-friendly target used by the course. |

The original `rental_count` is kept so learners can read predictions as real bike counts. The scaled target is included because smaller numbers are easier to use when teaching gradient descent by hand.

## Recreate the full CSV

From the repository root:

```bash
python scripts/prepare_bike_rentals_mini.py
```

No pandas or NumPy dependency is required. The script uses only the Python standard library.
